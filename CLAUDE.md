# CLAUDE.md

This project uses the cross-tool [`AGENTS.md`](AGENTS.md) convention for AI agent
guidance. **Read [`AGENTS.md`](AGENTS.md) first** — it is the source of truth for
what Relay is, its two topologies, where code and config live, the config/PII
split, and the design guardrails you must not violate.

If you are helping a user **install, deploy, configure, or operate** Relay, use
the paste-ready task runbooks in [`prompts/`](prompts/) (start at
[`prompts/README.md`](prompts/README.md)) rather than improvising from source.

Claude Code slash-command versions of those runbooks live in
[`.claude/commands/`](.claude/commands/).
